# Palette Milker Keyboard Shortcuts Guide

Palette Milker offers a comprehensive set of keyboard shortcuts to improve your workflow and make color palette creation more efficient. This guide documents all available shortcuts organized by category.

## Application Controls

| Shortcut | Description |
|----------|-------------|
| `Ctrl+Q` | Quit the application |
| `F1` | Show help screen |
| `D` | Toggle dark/light mode |
| `Tab` | Navigate to next field |
| `Shift+Tab` | Navigate to previous field |
| `Escape` | Go back to previous screen |

## Screen Navigation

| Shortcut | Description |
|----------|-------------|
| `1` | Switch to palette view |
| `2` | Switch to color picker |
| `3` | Switch to export options |

## Palette Management

| Shortcut | Description |
|----------|-------------|
| `Ctrl+S` | Save palette |
| `Ctrl+N` | Create new palette |
| `Ctrl+O` | Import palette |
| `Ctrl+E` | Export palette |
| `R` | Rename current palette |
| `C` | Copy/duplicate palette |

## Color Operations

| Shortcut | Description |
|----------|-------------|
| `A` | Add new color |
| `D` | Delete selected color |
| `E` | Edit selected color |
| `Ctrl+C` | Copy color to clipboard |

## Color Adjustment (Color Picker)

| Shortcut | Description |
|----------|-------------|
| `Up Arrow` | Increase hue |
| `Down Arrow` | Decrease hue |
| `Left Arrow` | Decrease saturation |
| `Right Arrow` | Increase saturation |
| `Shift+Up` | Increase lightness |
| `Shift+Down` | Decrease lightness |

## History Controls

| Shortcut | Description |
|----------|-------------|
| `Ctrl+Z` | Undo last action |
| `Ctrl+Shift+Z` | Redo last undone action |

## Display Options

| Shortcut | Description |
|----------|-------------|
| `H` | Toggle hex value display |
| `Space` | Toggle color details |

## Modal Screens

| Shortcut | Description |
|----------|-------------|
| `Escape` | Close modal screen |
| `Enter` | Confirm/accept (on dialogs) |

## How Key Bindings Work

Palette Milker uses Textual's binding system which allows for:

- App-wide bindings that work across all screens
- Screen-specific bindings for specialized functionality
- Context-sensitive actions based on your current task

All bindings are standardized across the application for consistency, with specialized screens adding only the bindings relevant to their functionality.

## Customizing Shortcuts

Currently, keyboard shortcuts cannot be customized by users. Future versions may include this feature.
